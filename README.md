# DOM Explorer - Task Management Dashboard

## Overview

DOM Explorer is a task management dashboard developed using HTML, CSS, and JavaScript. The project focuses on practical implementation of DOM manipulation concepts, event handling, event delegation, event propagation, and dynamic user interface updates.

The application allows users to create, edit, delete, search, and categorize tasks while maintaining real-time task statistics. It also includes demonstrations of advanced JavaScript concepts such as event bubbling, event capturing, and the difference between HTML attributes and DOM properties.

## Features

* Create new tasks with title, description, and category
* Edit existing tasks
* Delete tasks
* Mark tasks as completed or pending
* Search tasks by title
* Filter tasks by category
* Real-time task statistics
* Dark and light theme switching
* Modal-based task editing interface
* Event Delegation implementation
* Event Propagation (Capturing and Bubbling) demonstration
* Attributes vs Properties demonstration

## Key JavaScript Concepts Implemented

### DOM Manipulation

* createElement()
* append()
* prepend()
* setAttribute()
* dataset
* classList

### Event Handling

* Click Events
* Keyboard Events
* Change Events

### Event Delegation

A single event listener is attached to the task container to manage edit, delete, and complete actions for dynamically created task elements.

### Event Propagation

The project demonstrates:

* Event Bubbling
* Event Capturing

### Attributes vs Properties

The application includes an example showing the difference between HTML attributes and DOM properties using input elements.
